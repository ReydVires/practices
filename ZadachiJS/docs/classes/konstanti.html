konstanti Class

Defined in: TurboGjorgi.js:193
Module: TurboGjorgi

Sodrzhi funkcii-konstanti so koi se vrakja vrednost na dadena konstanta navedena pogore. JavaSkript nema konstanti pa zatoa se koristat funkcii za da se zachuva vrednosta na dadena promenliva. Isto taka se koristi funkciski blok(scope) bidejkji promenliva definirana vo funkcija so var e nedostapna nadvor od nea.
