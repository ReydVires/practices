percent

()

Defined in calc.js:321

Percent method. It needs both "result" and display field values to be different than 0 to work. Therefore an aritmetic operation must be used before you can calculate percents of a given value. Example: Press number 555 and then press the plus(+) sign button. "result" operand now has the value of 555 and the application waits for input for the second operand. Insert value of 33 and then press the percent(%) button. The application finds there are two available operands so it calculates 33% of 555 which is 183.15 and sets the display field to this new percent value. If you press the equal(=) sign button, it will sum up 555 + 183.15 which is 738.15. This same set of tasks applies to all other binary operations so if we used division(/) instead of adtion(+) the final result would be 3.030303 or (555 / (33% of 555)) = (555 / 183.15)
