Constants Class

Defined in: calc.js:180
Module: APP

Constants is a contructor so it should be called with new keyword. Each object should have its own Constants object. Creates object of four functions, set(), isDefined(), list() and get() which enable creation of new immutable constant that cannot be changed with Firebug or other debugging tool.
